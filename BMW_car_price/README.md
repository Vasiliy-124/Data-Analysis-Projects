# Предсказание стоимости автомобилей BMW

## О проекте
Данный репозиторий содержит учебный проект, выполненный в рамках программы «Машинное обучение» на факультете прикладной математики и информатики института МИРЭА. Работа посвящена решению задачи регрессии на реальных данных аукциона B2B.

## Задание
Основная цель работы — построить линейную модель для прогнозирования цены автомобиля и улучшить её качество с помощью методов регуляризации. В процессе выполнения необходимо было подготовить данные, обучить модель, подобрать гиперпараметры и оценить результат.

## Планы
Перед началом работы были поставлены следующие задачи:
1. Провести первичный анализ данных, обработать пропуски и удалить неинформативные признаки.
2. Создать новые признаки на основе других данных
3. Подготовить данные: выполнить кодирование категориальных переменных, масштабирование признаков и логарифмирование целевой переменной.
4. Построить линейную модель, применив регуляризацию (ElasticNet) и подобрать оптимальные гиперпараметры через кросс-валидацию.
5. Оценить качество модели на тестовой выборке с помощью метрики R².

## Результаты
В ходе работы был реализован полный пайплайн машинного обучения. Базовая линейная модель была дополнена регуляризацией ElasticNet. С помощью GridSearchCV были найдены оптимальные параметры: alpha = 0.001, l1_ratio = 1.0 (модель свелась к Lasso-регуляризации).

Итоговая метрика R² на тестовой выборке составила 0.8106. Это означает, что модель объясняет более 81% дисперсии цен, что является устойчивым результатом для линейного подхода на данном наборе данных. Все этапы предобработки и обучения были успешно выполнены в рамках поставленного задания без изменения базовой архитектуры модели.
