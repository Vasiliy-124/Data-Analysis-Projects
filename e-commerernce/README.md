# Анализ клиентской базы e-commerce

Анализ клиентской базы интернет-магазина с использованием RFM-анализа и кластеризации для сегментации клиентов и выявления поведенческих паттернов.

## Бизнес-задача

Сегментация клиентской базы и выявление факторов оттока для оптимизации маркетинговых стратегий и улучшения удержания клиентов.

## Данные

- Датасет E-commerce Sales: 34,500 транзакций
- 17 признаков: заказы, клиенты, товары, доставка, возвраты
- [Ссылка на данные](https://www.kaggle.com/datasets/miadul/e-commerce-sales-transactions-dataset/data)

## Методология

- RFM-анализ (Recency, Frequency, Monetary)
- Кластеризация K-Means для сегментации клиентов
- Анализ временных трендов и географического распределения
- Исследование факторов оттока и возвратов

## Результаты

- Выявлено 5 ключевых сегментов клиентов: VIP, Постоянные, Новые, Спящие, Потерянные
- Обнаружены критические проблемы: долгая доставка и высокие возвраты
- Определены приоритетные группы для удержания: Спящие, Новые, Потерянные клиенты
- Сформированы рекомендации по оптимизации логистики и скидочной политики

## Технологии

Python, pandas, scikit-learn, matplotlib, seaborn
