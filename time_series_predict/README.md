# Прогнозирование временного ряда

## О проекте
Учебный проект, выполненный в рамках программы «Машинное обучение» на факультете прикладной математики и информатики института МИРЭА. Работа посвящена прогнозированию временного ряда с использованием двух подходов: классического ARIMA и регрессионной модели

## Задание
Провести визуальный анализ и декомпозицию временного ряда, затем предсказать последние 24 значения двумя способами:
1. Как временной ряд (ARIMA)
2. Как задачу регрессии с инженерными признаками (CatBoost)

Оба решения проверить на кросс-валидации.

## Планы
1. Загрузить данные и провести первичный визуальный анализ ряда.
2. Выполнить декомпозицию ряда на тренд, сезонность и остаток.
3. Реализовать модель ARIMA с подбором гиперпараметров.
4. Сгенерировать временные признаки (час, день недели, месяц, часть суток) и обучить CatBoost с RandomizedSearchCV.
5. Сравнить модели по метрике MAE на тестовой выборке.

## Результаты
- **ARIMA(3,1,3)**: MAE = 953.00
- **CatBoost** (с признаками времени): MAE = 1650.09

Оба подхода показали работоспособность. Модель ARIMA превзошла CatBoost на тесте, что характерно для рядов с выраженной сезонностью и отсутствием сложного тренда. Декомпозиция подтвердила аддитивную природу сезонных колебаний, что обосновывает выбор модели.
