# Data Analysis Projects

Набор аналитических проектов, демонстрирующих применение методов анализа данных для решения бизнес-задач в области электронной коммерции.

## Описание проектов

| № | Проект | Описание | Инструменты |
|---|--------|----------|-------------|
| 1 | [Анализ клиентской базы](https://github.com/Vasiliy-124/Data-Analysis-Projects/tree/main/e-commerernce) | Сегментация клиентов и анализ поведенческих паттернов на основе синтетических данных e-commerce для оптимизации маркетинговых стратегий | pandas, numpy, matplotlib, seaborn, scikit-learn |
| 2 | [Статистический анализ A/B теста](https://github.com/Vasiliy-124/Data-Analysis-Projects/tree/main/AB-test) | Статистический анализ A/B теста трех промо-кампаний для выявления оптимальной стратегии увеличения продаж нового продукта | pandas, numpy, matplotlib, seaborn, SciPy, Statsmodels |
| 3 | [Анализ корзины покупок](https://github.com/Vasiliy-124/Data-Analysis-Projects/tree/main/Market_Basket_Analysis) | Анализ корзины покупок с выявлением товарных ассоциаций для повышения эффективности кросс-продаж и мерчендайзинга | pandas, numpy, matplotlib, seaborn, mlxtend |

## Ключевые компетенции

- **Анализ клиентского поведения**: сегментация, анализ жизненного цикла клиента
- **Статистический анализ**: проверка гипотез, A/B тестирование, анализ значимости
- **Машинное обучение**: кластеризация, ассоциативные правила (FP-Growth)
- **Визуализация данных**: создание информативных дашбордов и графиков
- **Работа с данными**: предобработка, очистка, feature engineering
