# Market Basket Analysis

Анализ корзины покупок для выявления товарных ассоциаций в данных онлайн-ритейла. Проект демонстрирует применение алгоритма FP-Growth для обнаружения паттернов совместных покупок.

## Бизнес-задача

Выявление товаров, которые часто покупаются вместе, для улучшения кросс-продаж, оптимизации мерчендайзинга и формирования промо-наборов.

## Данные

- Датасет Online Retail: 541,909 транзакций
- Период: декабрь 2010 - декабрь 2011  
- Атрибуты: транзакции, товары, цены, клиенты
- [Ссылка на данные](https://drive.google.com/file/d/1xshbSHBDQpTkAHKr6hdQ2D-N5KNgaYZ4/view?usp=share_link)

## Методология

- Предобработка: очистка от возвратов и одиночных покупок
- Алгоритм FP-Growth для поиска частых наборов товаров
- Ассоциативные правила с оценкой по метрикам Support, Confidence, Lift
- Критерии: Lift > 1.5, Confidence > 0.3, Support > 0.03

## Результаты

- Выявлено 28 сильных ассоциативных правил
- Наибольшая сила связи (Lift = 18.4) между чайными парами разных цветов
- Обнаружены устойчивые товарные кластеры для создания наборов

## Технологии

Python, pandas, matplotlib, seaborn, mlxtend
