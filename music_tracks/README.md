# Кластеризация музыкальных треков

## О проекте
Данный репозиторий содержит учебный проект, выполненный в рамках программы «Машинное обучение» на факультете прикладной математики и информатики института МИРЭА. Работа посвящена исследованию возможностей бессупервизного обучения для группировки музыкальных композиций по их акустическим характеристикам.

## Задание
Основная цель работы — применить алгоритмы кластеризации (KMeans и DBSCAN) к набору аудиопризнаков треков и оценить, насколько полученные группы соответствуют традиционной жанровой классификации. В процессе требовалось подготовить данные, подобрать оптимальные параметры моделей и рассчитать метрики качества относительно истинных меток.

## Планы
Перед началом работы были поставлены следующие задачи:
1. Провести первичный анализ данных и отобрать числовые аудиохарактеристики для обучения.
2. Выполнить предобработку: масштабирование признаков с помощью StandardScaler и кодирование жанровых меток.
3. Подобрать гиперпараметры: использовать метод локтя и силуэтный коэффициент для KMeans, проанализировать влияние параметров `eps` и `min_samples` для DBSCAN.
4. Обучить финальные версии моделей и оценить их качество относительно истинных жанров с помощью метрик ARI и AMI.
5. Сформулировать выводы о применимости выбранных методов для разделения музыкальных треков по жанрам.

## Результаты
В ходе работы был реализован полный пайплайн бессупервизного обучения. Наилучший результат показал алгоритм KMeans при количестве кластеров, равном 6: метрика ARI составила 0.057, AMI — 0.097. Модели DBSCAN продемонстрировали значения, близкие к нулю, из-за высокой вариативности данных в выбранном пространстве признаков.

Низкие значения метрик указывают на слабую корреляцию между стандартными аудиохарактеристиками и истинными жанровыми метками. Алгоритмы успешно выявляют группы треков со схожим звучанием, однако эти кластеры не совпадают с общепринятой жанровой таксономией.
