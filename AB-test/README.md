# A/B тестирование маркетинговых кампаний

Статистический анализ A/B теста для определения наиболее эффективной промо-акции, стимулирующей продажи нового продукта в сети быстрого питания.

## Бизнес-задача

Выбор оптимальной маркетинговой стратегии из трех промо-акций для максимизации продаж нового продукта.

## Данные

- Датасет Fast Food Marketing: 548 наблюдений
- Период: 4 недели тестирования
- Атрибуты: промо-акции, продажи, размер рынка, возраст магазина
- [Ссылка на данные](https://www.kaggle.com/datasets/chebotinaa/fast-food-marketing-campaign-ab-test)

## Методология

- Разведочный анализ распределения продаж по акциям
- ANOVA тест для сравнения средних across групп
- Попарное сравнение промо-акций (Tukey HSD)
- Анализ влияния размера рынка и возраста магазина

## Результаты

- Выявлена статистически значимая разница между промо-акциями
- Promotion 1 показала наивысшие средние продажи
- Promotion 3 может рассматриваться как рентабельная альтернатива
- Сформированы рекомендации по выбору маркетинговой стратегии

## Технологии

Python, pandas, scipy, statsmodels, matplotlib, seaborn
